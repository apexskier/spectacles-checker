# Spectacles Checker

Little script that checks if [@snap's](https://twitter.com/Snap) dropping a new
Snapbot anytime soon and lets you know with a notification via
[Pushover](https://pushover.net).

```sh
LAT={your latitude in degrees} \
LNG={your longitude in degrees} \
PUSH_TOKEN={get from pushover} \
PUSH_USER_KEY={get from pushover} \
node index.js
```
